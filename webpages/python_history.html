Python history

Python laid its foundation in the late 1980s. Guido Van Rossum started implementing Python in December 1989 at CWI in Netherland. In February 1991 Van Rossum officially published the code labelling it 0.9.0 to alt.sources.

Python 1.0 was released in 1994 implementing new feature likes lambda, map, filter and reduce. By come Python2.0 added new features such as listr comprehensions, garbage collection systems.

Python 3.0 also called Py3K was released on December 3, 2008. it was designed to rectify the fundamental flaw of the language.

Why the name Python?

Van Rossum wanted to select a name which is unique, sort and a little bit of mysterious. So he devided to select the name Python after the "Monty Python's Flying Circus"

The fact behind him choosing this name after a show is that, Van Rossum was the script of the popular BBC comedy series. The comedy series was creative and well random. It talked about everthing. Thus it's slow and unpredictable. whcih made it very intresting.

Python is also a very versatile and is widely used in every field from mobile application all the way to Artifical Intelligence and Machine Learning.
